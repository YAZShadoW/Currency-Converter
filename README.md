## Требования
- Python 3.x
- Библиотека Requests

## Установка зависимостей
1. Установите Python (если не установлен).
2. Установить библиотеку requests.
3. Выполните команду в папке проекта:
   ```
   pip install -r requirements.txt
   ```
4. Получите бесплатный API-ключ на сайте [exchangerate-api.com](https://www.exchangerate-api.com/).
5. Вставьте ключ в файл `converter.py` вместо `YOUR_API_KEY`.

## Запуск приложения
1. Откройте терминал в папке проекта.
2. Запустите:
   ```
   python main.py
   ```
3. Введите сумму в поле «Сумма», выберите валюты и нажмите «Конвертировать».
4. История операций сохраняется в файл `history.json` и отображается в таблице.

## Примеры использования и тесты
- **Корректный ввод:** Введите число (например, `100`), выберите разные валюты — результат появится ниже.
- **Некорректный ввод:** Введите текст или отрицательное число — появится сообщение об ошибке.
- **История:** После перезапуска приложения история загружается из файла автоматически.
- **Оффлайн:** Отключите интернет и попробуйте конвертировать — появится ошибка о недоступности API.
